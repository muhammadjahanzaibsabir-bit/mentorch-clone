  1. Is file ko `index.html` naam se save karo.
  2. File par right-click → Open with → Browser (Chrome/Edge/Firefox).
  3. Page khul jayega, "Start Chat" pe click karo aur chat try karo.

How to replace demo with real AI (short):

  1. Sign up for a free account on an AI provider (OpenAI, Hugging Face) and get an API key.
  2. Create a small backend (Node.js + Express) that stores the API key securely and forwards requests to the AI provider.
  3. Change the `sendMessageToAI()` function below to call your backend endpoint (not directly the API from browser).
  4. Deploy backend to free hosting (Render free, Vercel, Fly, Railway free tier) and frontend to GitHub Pages / Netlify / Vercel.
